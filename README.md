# MMObingo

1. Players follow a link to one (or a few) pre-made bingo board(s);
2. Players register with a name and email to play;
3. To occupy a square, users must submit a valid url;
4. Url's to repos are checked for date and username of last commit
5. progress for all players can be seen, including completed boards

there should be no ability for users to change the board
there should be no need for a chat function
